# 📊 Skill Gap Analysis Tool

A machine learning-powered web application that analyzes interns' skills and compares them with industry job requirements.

## 🚀 Features

- Analyze intern skill gaps
- TF-IDF based skill vectorization
- K-Means clustering of job roles
- Skill match percentage
- Matching and missing skills
- Recommended training courses
- Interactive Streamlit dashboard
- Plotly visualizations

## 🛠 Technologies

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
